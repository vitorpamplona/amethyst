# Changelog

The changelog has moved. Release notes now live in
[`docs/changelog/`](docs/changelog/), one file per version, with an index in
[`docs/changelog/README.md`](docs/changelog/README.md).

When cutting a new release, copy
[`docs/changelog/TEMPLATE.md`](docs/changelog/TEMPLATE.md) to a new
`vMAJOR.MINOR.PATCH.md` file (zero-padded minor/patch, e.g. `v1.11.00.md`) and
add it to the top of the index.
